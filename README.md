## 🚀 快速开始

### 环境要求

- Python 3.10+
- Git
- OpenAI API Key（或国内镜像）

### 一键安装

```bash
# 1. 克隆仓库
git clone https://github.com/zdnuist/ai-agent-tutorial.git
cd ai-agent-tutorial

# 2. 安装依赖
pip install -r requirements.txt

# 3. 配置 API Key
cp .env.example .env
# 编辑 .env 填入你的 API Key

# 4. 运行示例
python examples/01_simple_agent.py
```
